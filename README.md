# GR4AI website

Static site for https://gr4ai.org

## What's inside
- `index.html` — the whole page
- `assets/` — illustrations and team photos

## How it's deployed
This repo is connected to IONOS Deploy Now. Pushing to `main` auto-publishes to gr4ai.org.

## Editing
Edit `index.html` directly, commit, push. The site updates within a minute.
